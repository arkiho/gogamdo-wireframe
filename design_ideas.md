## 아이디어 1: "Architectural Noir" — 건축적 다크 미니멀리즘

**Design Movement:** Dark Minimalism + Architectural Brutalism

**Core Principles:**
1. 극단적 여백과 대비 — 딥 블랙/차콜 배경 위에 밝은 요소가 떠오르는 구조
2. 건축적 그리드 — 비대칭 그리드로 공간을 분할하되, 건축 도면처럼 정밀한 정렬
3. 물성(Materiality) 표현 — 콘크리트 텍스처, 메탈릭 골드 라인, 유리 반사 효과
4. 시네마틱 스토리텔링 — 각 섹션이 영화 장면처럼 전환

**Color Philosophy:** 차콜 블랙(#0D0D0D) 주 배경 + 브러시드 골드(#B8956A) CTA + 라이트 그레이(#E8E4E0) 본문

**Layout Paradigm:** 풀스크린 이미지 + 오버레이 텍스트. 비대칭 2/3 + 1/3 분할.

**Signature Elements:** 골드 라인 디바이더, 마스크 리빌 효과

**Animation:** GSAP 기반 시네마틱 전환 (0.8~1.2초)

**Typography:** Playfair Display + Pretendard

## 아이디어 3: "Precision Studio" — 정밀한 스튜디오

**Design Movement:** Swiss Design + Contemporary Editorial

**Core Principles:**
1. 수학적 그리드 — 12컬럼 그리드 엄격 준수 + 의도적 브레이크
2. 타이포그래피 주도 — 이미지보다 타이포그래피가 디자인을 이끔
3. 정보 밀도 조절 — 섹션마다 밀도 변화로 리듬감
4. 모노크롬 + 원 포인트 — 무채색 + 골드 원포인트

**Color Philosophy:** 퓨어 화이트(#FFFFFF) + 트루 블랙(#111111) 극단적 대비. 시그니처 골드(#C8A96E)가 유일한 액센트로 CTA와 핵심 숫자에만 사용.

**Layout Paradigm:** 에디토리얼 그리드. 대형 타이포그래피 + 정밀 크롭 이미지. 체스판 교차 배치.

**Signature Elements:** 오버사이즈 넘버링(200px+), 수직 프로그레스 바

**Animation:** CSS + Framer Motion. 텍스트 슬라이드업, 이미지 클립패스 리빌 (0.3~0.6초)

**Typography:** Sora(영문) + Pretendard(한글). Bold/Black 헤드라인 + Regular 본문.

## 선택: 아이디어 3 "Precision Studio" — 정밀한 스튜디오

**선택 이유:**
1. B2B 인테리어 회사에 가장 적합한 전문성과 신뢰감 전달

4. 에디토리얼 레이아웃이 포트폴리오 사진의 임팩트를 극대화
