## 2. Manus 내장 호스팅으로 배포하기

Manus 플랫폼은 내장 호스팅 서비스를 제공하며, 커스텀 도메인 연결도 지원합니다. 별도의 외부 호스팅(Vercel, Railway 등)을 사용할 필요가 없습니다.

### 2.1 체크포인트 저장 및 배포

1. **체크포인트 저장**: 모든 변경사항이 반영된 최신 체크포인트를 저장합니다.
2. **Publish 버튼 클릭**: Manus Management UI 우측 상단의 **Publish** 버튼을 클릭합니다.
3. **배포 완료 확인**: 배포가 완료되면 자동으로 `xxx.manus.space` 형태의 URL이 생성됩니다.

### 2.2 기본 도메인 설정

배포 후 기본적으로 `gogamdo-wireframe.manus.space` 형태의 도메인이 할당됩니다. Management UI의 **Settings > Domains** 패널에서 도메인 프리픽스를 원하는 이름으로 변경할 수 있습니다 (예: `kokamdo.manus.space`).

## 3. 커스텀 도메인 연결 (kokamdo.co.kr)

고감도 공식 도메인 `kokamdo.co.kr`을 연결하려면 다음 단계를 따릅니다.

### 3.1 Manus 도메인 설정

1. Management UI에서 **Settings > Domains** 패널을 엽니다.
2. **커스텀 도메인 추가**를 클릭합니다.
3. 사용할 도메인을 입력합니다: `kokamdo.co.kr` 또는 `www.kokamdo.co.kr`
4. 시스템이 안내하는 DNS 레코드 정보를 확인합니다.

### 3.2 DNS 설정 (도메인 등록 업체에서)

도메인 등록 업체(가비아, 카페24, AWS Route 53 등)의 DNS 관리 페이지에서 다음 레코드를 추가합니다.

| 레코드 유형 | 호스트 | 값 | TTL |
|-------------|--------|-----|-----|
| CNAME | www | Manus에서 안내하는 CNAME 값 | 300 |
| A 또는 CNAME | @ (루트) | Manus에서 안내하는 값 | 300 |

DNS 전파에는 최대 24~48시간이 소요될 수 있으나, 일반적으로 수 분 내에 적용됩니다.

### 3.3 SSL 인증서

Manus 호스팅은 커스텀 도메인에 대해 **자동으로 SSL 인증서를 발급**합니다. 별도의 인증서 구매나 설정이 필요하지 않습니다. 도메인 연결 후 HTTPS가 자동으로 활성화됩니다.

## 4. 이메일 도메인 인증 (Resend)

현재 이메일 발송 시 `kokamdo.co.kr` 도메인 인증이 필요합니다. Resend에서 도메인을 인증하면 `noreply@kokamdo.co.kr`로 이메일을 발송할 수 있습니다.

### 4.1 Resend 도메인 인증 절차

1. [Resend 대시보드](https://resend.com/domains)에 로그인합니다.
2. **Add Domain**을 클릭하고 `kokamdo.co.kr`을 입력합니다.
3. Resend가 안내하는 DNS 레코드를 도메인 등록 업체에 추가합니다.

| 레코드 유형 | 호스트 | 값 | 용도 |
|-------------|--------|-----|------|
| MX | 지정된 호스트 | Resend MX 값 | 메일 수신 |
| TXT | 지정된 호스트 | Resend SPF 값 | 발신자 인증 |
| CNAME | 지정된 호스트 | Resend DKIM 값 | 이메일 서명 |

4. DNS 레코드 추가 후 Resend에서 **Verify** 버튼을 클릭합니다.
5. 인증이 완료되면 `noreply@kokamdo.co.kr`로 이메일 발송이 가능합니다.

### 4.2 이메일 발송 대상

도메인 인증 후 다음 이메일들이 정상 발송됩니다.

| 이메일 유형 | 발신자 | 수신자 | 트리거 |
|-------------|--------|--------|--------|
| 회원가입 인증 | noreply@kokamdo.co.kr | 고객 | 고객 포털 회원가입 시 |
| 비밀번호 재설정 | noreply@kokamdo.co.kr | 고객 | 비밀번호 찾기 요청 시 |
| 리뷰 요청 | noreply@kokamdo.co.kr | 프로젝트 담당자 | 관리자가 리뷰 요청 생성 시 |
| AI 분석 보고서 | noreply@kokamdo.co.kr | 고객 | AI 보고서 생성 완료 시 |
| 뉴스레터 | noreply@kokamdo.co.kr | 구독자 | 관리자가 캠페인 발송 시 |
