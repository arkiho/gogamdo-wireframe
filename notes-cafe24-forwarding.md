# 카페24 도메인 포워딩 설정 방법

## 요약
- 카페24에서 구입한 도메인은 도메인 포워딩 서비스를 제공
- 먼저 네임서버를 카페24 포워딩 전용 네임서버로 변경해야 함
- 네임서버 변경 후 '도메인 포워딩 관리'에서 유동/고정 포워딩 설정
- 웹호스팅에 연결된 도메인은 포워딩 불가 → 먼저 연결 해제 필요
- https:// 형식 포워딩은 미지원, http://만 설정 가능

## 단계
1. 웹호스팅 도메인 연결 해제: 나의 서비스 관리 > 호스팅관리 > 기본관리 > 도메인 연결관리
2. 네임서버 변경: 나의 서비스 관리 > 도메인관리 > 네임서버 변경 → 포워딩 전용 네임서버로 변경
3. 도메인 포워딩 관리에서 목적지 URL 설정
4. 유동 포워딩: 주소창이 목적지 주소로 변경됨
5. 고정 포워딩: 주소창에 원래 도메인 유지

## 주의사항
- https 포워딩 미지원 → 이 방법은 Manus 연결에 적합하지 않을 수 있음
- 대안: DNS CNAME 레코드 직접 설정이 더 적합
