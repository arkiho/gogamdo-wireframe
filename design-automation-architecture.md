# 고감도 인테리어 설계 자동화 시스템 아키텍처

## 전체 파이프라인 개요

```
고객 → 도면 업로드 → RFP 수집 → AI 분석 → 레이아웃 생성 → 렌더링 → 투어 영상 → 제안서 → 견적서 → 첫 미팅 계약
```

## 1단계: 프로젝트 생성 및 도면 업로드
- PDF/이미지 도면 업로드 (S3 저장)
- AI 도면 분석: 면적 추출, 벽체/기둥/창문 인식, 공간 구조 파악
- 도면 메타데이터 자동 추출 (총 면적, 층수, 방 개수 등)

## 2단계: RFP 수집 (3가지 방식)
### A. 직접 입력 폼 (구조화된 설문)
### B. AI RFP 생성기 (대화형)
### C. 컨설팅 챗봇 (자유 대화)

### RFP 핵심 항목 (리서치 기반):

#### 기본 정보
- 회사명, 업종, 설립연도
- 담당자 이름, 직책, 연락처, 이메일
- 현재 사무실 주소 / 신규 사무실 주소
- 프로젝트 유형: 신규 입주 / 이전 / 리모델링

#### 공간 요구사항
- 총 면적 (㎡ 또는 평)
- 현재 인원수 / 채용 예정 인원 (1년/3년)
- 부서 구성 (부서명, 인원수, 업무 특성)
- 필요 공간 유형:
  * 개인 업무공간 (고정석/자유석/핫데스크)
  * 임원실 (개수, 규모)
  * 회의실 (소/중/대, 개수)
  * 컨퍼런스룸/세미나실
  * 라운지/휴게공간
  * 카페테리아/주방
  * 전화부스/집중업무실
  * 서버룸/IT실
  * 창고/수납공간
  * 리셉션/로비
  * 수유실/기도실
  * 기타 특수 공간

#### 디자인 선호도
- 선호 스타일: 모던/미니멀/내추럴/인더스트리얼/클래식/기타
- 브랜드 컬러 / 기업 CI
- 브랜드 가이드라인 파일 업로드
- 참고 이미지/레퍼런스 URL
- 선호하는 마감재 (목재/콘크리트/유리/금속 등)
- 조명 선호도 (자연광 활용, 간접조명, 조도 수준)

#### 기능 요구사항
- AV/IT 장비 요구사항
- 네트워크 인프라 (유선/무선/서버)
- 보안 시스템 (출입통제, CCTV)
- 음향 프라이버시 수준
- 냉난방/환기 요구사항
- 친환경/ESG 요구사항

#### 예산 및 일정
- 총 예산 범위 (설계비 포함/별도)
- 우선순위: 디자인 > 기능 > 예산 / 예산 > 기능 > 디자인 등
- 희망 착공일
- 희망 완공일 (입주 예정일)
- 공사 중 사무실 사용 여부

#### 기타
- 건물 관리 규정/제약사항
- 기존 가구 재활용 여부
- 특별 요청사항
- 경쟁사/벤치마크 사무실

## 3단계: AI 데이터 분석 및 레이아웃 생성
- RFP + 도면 데이터 통합 분석
- 공간 프로그래밍 (면적 배분 최적화)
- 동선 분석 및 인접성 매트릭스
- 2~3개 레이아웃 옵션 제안
- 각 옵션별 장단점 비교

## 4단계: AI 렌더링 생성
- 주요 공간별 포토리얼리스틱 렌더링
  * 리셉션/로비
  * 오픈 오피스
  * 회의실
  * 임원실
  * 라운지/휴게공간
  * 카페테리아
- 스타일/마감재 반영
- Before/After 비교

## 5단계: 투어 영상 생성
- 렌더링 이미지 기반 워크스루 영상
- AI 비디오 생성 활용

## 6단계: 제안서 자동 생성
- 고객사 분석 (업종, 규모, 문화)
- 설계 컨셉 설명
- 레이아웃 도면
- 렌더링 이미지
- 공간별 설명
- 마감재/가구 제안
- 프로젝트 일정표
- 고감도 소개 및 차별점

## 7단계: 견적서 자동 생성
- 공종별 상세 견적
- 자재 수량 산출
- 단가 적용 (학습 데이터 기반)
- 총 공사비 산출
- 옵션별 견적 비교
