# OpsX 프로세스 정보 (opsx.co.kr)

## 핵심 메시지
- 사무환경 컨설팅부터 원스톱 구축까지
- 데이터 기반 진단으로 현재의 문제를 명확히 파악
- 전담 매니저가 설계부터 시공까지 최적의 오피스 완성

## 인사이트 서비스 4대 가치
1. 데이터 기반 심층 분석 - 직원 목소리를 정밀 수치화
2. 맞춤형 공간 전략 수립 - 조직 문화와 업무 특성 고려
3. 최종 의사결정 보고서 - ROI 분석 포함 전문 진단 보고서
4. 실무용 RFP 패키지 - 과업지시서 생성

## 8단계 프로세스
1. 진단 설계 및 맞춤화 - 맞춤형 진단 문항 설계
2. 스마트 웹 설문 조사 - 업무 스타일, 공간 사용 패턴 수집
3. 전문가 현장 실사 - 공간 동선, 노후도, 현장 제약 분석
4. 심층 인터뷰 (FGI) - 핵심 그룹 인터뷰로 정성적 인사이트
5. AI 기반 데이터 분석 - 빅데이터 분석 알고리즘
6. 전략적 진단 리포트 - 점수화 + 구체적 공간 개선안
7. 1:1 전략 컨설팅 - 예산/상황 맞춤 실행 로드맵
8. 사후 관리 및 측정 - 개선 효과 측정, 운영 최적화
